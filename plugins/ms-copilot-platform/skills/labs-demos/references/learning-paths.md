# Learning Paths

Structured multi-lab curricula for different audiences learning the Microsoft Frontier Agent Platform.

## Path 1: Citizen Developer Track (2-3 Days)

**Audience:** Business users, power users, citizen developers
**Goal:** Build and deploy agents using low-code/no-code tools

### Day 1: Foundations
| Time | Lab | Focus |
|------|-----|-------|
| 9:00-9:30 | Platform Overview | Architecture walkthrough |
| 9:30-10:00 | Lab GS-1: SharePoint Agent | First agent in 15 minutes |
| 10:00-10:30 | Lab GS-2: Agent Builder | Custom instructions and knowledge |
| 10:30-12:00 | Lab CS-1: Copilot Studio Basics | Visual agent designer |
| 1:00-3:00 | Lab CS-2: Connectors & Actions | Business process automation |
| 3:00-4:00 | Lab GOV-1: Agent 365 Basics | Registration and governance |

### Day 2: Advanced Low-Code
| Time | Lab | Focus |
|------|-----|-------|
| 9:00-11:00 | Lab CS-3: Autonomous Agent | Proactive agent behavior |
| 11:00-12:00 | Lab INT-2: Enterprise Connectors | CRM integration |
| 1:00-3:00 | Custom Project | Build an agent for your team |
| 3:00-4:00 | Review & Next Steps | Feedback, Q&A, learning resources |

### Day 3 (Optional): Integration & Scale
| Time | Lab | Focus |
|------|-----|-------|
| 9:00-11:00 | Lab INT-1: Fabric Data Agent | Data intelligence |
| 11:00-12:00 | Lab GOV-2: Monitoring | Copilot Analytics |
| 1:00-3:00 | Custom Project (continued) | Enhance your agent |
| 3:00-4:00 | Presentation | Demo your agent |

## Path 4: Full Stack Track (5-7 Days)

**Audience:** Teams wanting comprehensive platform knowledge
**Goal:** Complete understanding from citizen dev to enterprise architecture

Combines all three tracks:
- Days 1-2: Citizen Developer Track (condensed)
- Days 3-5: Pro Developer Track
- Days 6-7: Architect Track

## Certification Alignment

These learning paths align with Microsoft certifications:
- **MS-600**: Building Apps and Solutions with M365 Core Services
- **AI-102**: Designing and Implementing a Microsoft Azure AI Solution
- **PL-200**: Microsoft Power Platform Functional Consultant
- **PL-600**: Microsoft Power Platform Solution Architect

## Custom Learning Path Generation

When a user requests a custom learning path:
1. Assess current knowledge level
2. Identify specific learning objectives
3. Determine available time
4. Select and sequence appropriate labs
5. Add architecture context between labs
6. Include hands-on project time
7. Plan assessment/review checkpoints
