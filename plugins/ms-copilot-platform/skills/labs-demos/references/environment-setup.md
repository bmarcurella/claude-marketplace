# Lab Environment Setup

Prerequisites and setup instructions for running hands-on labs.

## Minimum Requirements

### For All Labs
- Microsoft 365 developer tenant (free: https://developer.microsoft.com/en-us/microsoft-365/dev-program)
- Web browser (Edge or Chrome recommended)
- Internet connection

### For Copilot Studio Labs
- Copilot Studio license (trial available)
- Power Platform environment

### For Declarative Agent Labs
- VS Code with Microsoft 365 Agents Toolkit extension
- Node.js 20+
- M365 Copilot license on the dev tenant

### For Custom Engine Agent Labs
- VS Code with Agents Toolkit
- Python 3.12+ (or .NET 8+ / Node.js 20+)
- Docker Desktop
- Azure subscription (free trial works)
- Azure CLI

### For Foundry Labs
- Azure subscription with AI services access
- Azure CLI with ml extension

### For Governance Labs
- M365 admin access on dev tenant
- Agent 365 license (trial available)
