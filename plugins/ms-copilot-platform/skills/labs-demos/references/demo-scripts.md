# Demo Scripts

Pre-built demo narratives for showcasing agent capabilities to stakeholders.

## Demo 1: The 2-Minute Agent (SharePoint Agent)
**Audience:** Executives, business stakeholders
**Duration:** 2 minutes
**Script:**
1. Open SharePoint document library with company policies
2. Click Copilot → Create Agent
3. Ask: 'What is our remote work policy?'
4. Show the agent citing the specific policy document
5. Key message: 'From documents to intelligent Q&A in 2 minutes'

## Demo 2: Business Process Agent (Copilot Studio)
**Audience:** Business analysts, department leads
**Duration:** 10 minutes
**Script:**
1. Show the Copilot Studio visual designer
2. Demonstrate a pre-built agent that handles IT helpdesk requests
3. Show: user asks about password reset → agent looks up user → creates ticket → sends confirmation
4. Show the connector to ServiceNow
5. Show autonomous trigger: agent proactively reminders for open tickets
6. Key message: 'Agents that don't just answer — they act'

## Demo 3: Custom Engine Agent (Agents SDK)
**Audience:** Developers, architects
**Duration:** 15 minutes
**Script:**
1. Show the project structure in VS Code
2. Walk through the agent code (ActivityHandler)
3. Show tool definitions (Semantic Kernel plugins)
4. Demo in Agents Playground
5. Show multi-tool execution (agent calls 3 tools in sequence)
6. Show MCP server integration
7. Key message: 'Full control with enterprise-grade infrastructure'

## Demo 4: Multi-Agent System (A2A)
**Audience:** Enterprise architects
**Duration:** 20 minutes
**Script:**
1. Show architecture diagram (router + specialist agents)
2. Demo: user asks compound question spanning HR and IT
3. Show router delegating to HR Agent then IT Agent via A2A
4. Show aggregated response
5. Show Agent 365 governance dashboard
6. Key message: 'Specialized agents that collaborate like a team'
