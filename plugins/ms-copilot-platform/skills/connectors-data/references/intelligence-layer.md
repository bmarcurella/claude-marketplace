# Intelligence Layer Guide

Work IQ, Fabric IQ, and Foundry IQ — the AI services that make agents smarter.

## Work IQ
Workplace intelligence built on Microsoft Graph signals.

### Capabilities
- **Work memory** — Remembers user preferences and past interactions
- **Personalization** — Adapts responses based on user role and context
- **Collaboration signals** — Understands team dynamics and communication patterns

### Use Cases
- Agent personalizes responses based on user's department and role
- Agent surfaces relevant documents based on recent work activity
- Agent adjusts communication style based on user preferences

## Fabric IQ
Data intelligence for enterprise analytics.

### Capabilities
- **Semantic understanding** — Translates natural language to data queries
- **Data discovery** — Finds relevant datasets across the organization
- **Insight generation** — Automatically identifies trends and anomalies

### Use Cases
- Agent answers business questions by querying semantic models
- Agent generates data-driven reports on demand
- Agent alerts on anomalies in business metrics

## Foundry IQ
AI model intelligence and orchestration.

### Capabilities
- **Model routing** — Selects the best model for each task
- **Orchestration** — Manages multi-step AI workflows
- **Evaluation** — Measures quality and safety of AI outputs

### Use Cases
- Agent automatically uses cheaper model for simple queries
- Agent routes complex reasoning to more capable models
- Agent evaluates its own responses for quality before sending
